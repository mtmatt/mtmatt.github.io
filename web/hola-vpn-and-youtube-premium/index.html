使用 Hola VPN 從土耳其跨區訂閱 YouTube Premium

發表於 2023-03-20 分類於 web
文章字數： 1.5k 所需閱讀時間 ≈ 3 分鐘

前言

YouTube 上的廣告似乎越來越多，電腦版上的 YouTube 廣告，找支 Adblock 就可以搞定，但，我還有 Apple TV 與 Google Chromecast(支援 Google TV)，要無廣告順順、爽爽觀賞，就有點麻煩。

花錢，購買 YouTube Premium 就成為比較容易處理的方式。

然而，要以相對划算的價格達到無廣告的目的，網路上提供的解決之道，關鍵就在於使用 VPN 軟體，跨區到相對比較便宜的國家進行購買訂閱的動作，而本篇貼文，主要就是我嘗試爬文後，參考做法後的心得，以及現階段比較可行的方式與實做過程。

多數 VPN 要註冊、要安裝獨立程式、要錢

  1. 要註冊：比方說，我自己在使用 Windscribe 的時候，發現還得先註冊一個帳號才給用，…也太累。

  2. 要安裝獨立程式：要另外安裝獨立程式，對我來說，並不是很棒的選擇，倒不如直接在瀏覽器上面安裝擴充功能或插件就可以使用，來得方便。

  3. 要錢：為了訂閱便宜的 YouTube Premium 想方設法，找到方法後，卻要先花一筆錢訂閱 VPN 服務，那我不如先繼續花一點點時間，看看有沒有免費的方案。除非，免費的讓我花超過 1 小時以上，那我就會考慮在 3 美元以內訂閱有效的 VPN 服務，嘗試達成效果。簡單說，非不得已，我是不會想要先花錢訂閱 VPN 服務的。

不用註冊、不用安裝獨立程式、免費的處理方式

  1. 使用前的提醒：由於 Google 似乎會擋走 VPN 去收費相對便宜的國家訂閱的使用者，因此，今天介紹的 VPN 是我確定還可以使用的，但並不保證一定持續可用，所以，下單前請放亮眼睛，別賠了夫人又折兵。下圖所示，就是使用 Windscribe 走到土耳其，卻被收加幣(CA) 的例子。

  2. Hola VPN：截至發文前，我親自跑一趟，確定 Hola VPN 仍是可行的。

安裝使用 Hola VPN 與訂閱 YouTube Premium

  1. Hola VPN 讓我欣嘗的地方，就是它能夠透過 Google Chrome 的「擴充功能」安裝進來。本次演示，我則是選擇在 Microsoft Edge 完成。

  2. 安裝 Hola VPN：步驟如下圖串。

    (↓)點 Microsoft Edge 工具列的「．．．」(對，就是這個在微軟近幾年的系統介面裡，很常看到的「點點點」)。

    (↓)點「開啟 Microsoft Edge 外掛程式網站」。

    (↓)搜尋關鍵字「hola」。

    (↓)找到「Hola Free VPN Proxy Unblocker - Best VPN」，點按旁邊的「取得」鈕。

    (↓)安裝完成的畫面，我會建議從「擴充功能」裡面，設定為「在工具列顯示」，…因為等一下就會用到，顯示出來我自己是覺得比較方便做事。

  3. 進入 YouTube 網站後，啟動 Hola VPN。

  4. 承上圖，點一下中間國旗旁的「▼」(倒三角)，搜尋關鍵字「turkey」，下圖是我打到「tur」就已經把 Turkey 找出來，記得點一下找出來的「Turkey」。

  5. 走到這邊，都沒有問題的話，就可以看到 Turkey 顯示成彩色，這樣的話，大致就沒有問題。

  6. 找到網頁左手邊的「YouTube Premium」，並且直接給它點下去。

  7. 如果更面顯示 TRY 29.99(截稿前是這個價格)，基本上就沒有問題。

後記

  1. 當我看到「TRY」字樣，雖然知道它指的是「新土耳其里拉」，仍不免覺得 Google 似乎有玩梗的嫌疑。
  2. 入手價格：我下單的時候，花費 491 元新台幣(不含信用卡手續費)，再怎麼算都划算。
