免費產出自己網站上的 Favicon

發表於 2008-03-09 分類於 web
文章字數： 849 所需閱讀時間 ≈ 2 分鐘

製作 Favicon 在老早以前，讀過部分參考文件，總讓人覺得似乎是件難事。

我曾經給部落格做出一個，猶記得當時是先使用 Flash 在線條上花點心思，製作出一個小小的公雞頭圖像，再來的程序我有點沒印象，好像是交給 Fireworks 處理後將 favicon.png 易名為 favicon.ico ，最後放到網站上，沒想到就這麼完成 (( 很神奇，卻不是很好的方法，在此仍建議使用專門製作 .ico 的 ICON 軟體。

針對 Windows 與 Mac ，我分別推薦 Microangelo 與 Iconbuilder ，有興趣的伙伴可以再參考 Icon & Cursor Editors for Windows 與 Icon & Cursor Editors for Macintosh ，比較抱歉的是未曾在 Linux 上製作過的我，得請習慣拿 Linux 當桌面系統的伙伴自個兒挖一下 Google 。 ))。

今兒個我突然想再做一個，用 Google 很快地就挖出符合需求的資源，更重要的是製作過程無痛且快速。

多家比較後，我認為比較優秀的轉檔網站是 FavIcon from Pics – favicon.ico for your website ，因為解析高的圖案經過轉換程序後所展現出的 favicon.ico 仍舊美美地，不會有過份的失真。

點下 [ Download Favicon ] 會得到封裝好的壓縮檔，解開並將 favicon.ico 取出即可服用，壓縮檔裡頭有個 [ extra ] 資料夾，有興趣玩動態 Favicon 還得動點工，在此就不多提。最後，再把 favicon.ico 放到網站的根目錄裡就大功告成，突然之間，我感受到過高的自戀濃度，還望大伙海涵。
